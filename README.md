Изначально при запуске программа просит id камеры во избежание ошибок, а также расположение текстового файла в котором будут распологаться все значения. Затем у нас есть четыре окна между которыми можно переключаться с помощью Tab widget. Первое окно называется **'Оригинальное изображение с камеры'** оно выводит оригинальное изображение с камеры. Второе окно называется **'Gray'** выводит gray изображение.Третье **'Trackbar RGB'** в нем выводится маска изображение в которой изменяются минимальные и максимальные значения RGB. Четвертое окно называется **'Trackbar HSV'** и выполняет те же функции что и второе окно но со значениями HSV. Также на первом и втором окне находится кнопка 'вывести изображение' при нажатии которой появиться видеоизображение без масок, а при втором нажатии на эту же кнопку закрывает.
